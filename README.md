# Cryptography System

## Description

A simple Python-based cryptography system that allows users to encrypt and decrypt messages using a secret key through a menu-driven interface.

## Features

* Encrypt messages
* Decrypt messages
* User-friendly menu options
* Secret key input for encryption and decryption
* Exit option

## Technologies Used

* Python
* Functions
* String Manipulation

## How to Use

1. Run the Python program.
2. Choose one of the following options:

   * 1: Encrypt a message
   * 2: Decrypt a message
   * 3: Exit
3. Enter the message and the secret key.
4. View the encrypted or decrypted result.

## Project Purpose

This project was developed to practice Python programming concepts, including functions, user input handling, string processing, and basic cryptography techniques.
